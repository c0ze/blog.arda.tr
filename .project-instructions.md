- **Metadata:** Every route has unique Meta Titles and Descriptions via frontmatter `keywords` and `description` fields.
- **Sitemap:** Auto-generated on build via `scripts/generate-sitemap.ts`.
- **robots.txt:** Configured to allow all crawlers including AI bots (GPTBot, ClaudeBot, etc.).
